# 📊 HR Analytics Dashboard Using Excel

## 📌 Project Overview
Interactive HR Analytics Dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, KPI calculations, and data visualization techniques.

## 🎯 Business Objective

To build an interactive dashboard that enables HR managers to monitor employee performance, identify attrition patterns, and make informed workforce decisions.

---
## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas
- KPI Calculations

---

## 📂 Dataset

The dataset contains employee information, including:

- Emp ID
- Employee Name
- Department
- Gender
- Age
- Salary
- Performance Rating
- Attrition Status
- Joining Date

---

## 📈 Dashboard KPIs

- Total Employees
- Attrition Rate
- Average Salary
- Average Performance Rating
- Total Departments
- Male Employees
- Female Employees

---

## 📊 Dashboard Features

- Department-wise Employee Distribution
- Gender Diversity Analysis
- Attrition by Department
- Performance Rating Analysis
- Salary Distribution
- Department-wise Average Salary
- Interactive Slicers
- Dynamic KPI Cards

---

## 🔍 Key Insights

- Total workforce consists of **100 employees** across **5 departments**.
- HR department has the highest employee count (**23 employees**).
- Gender distribution is nearly balanced with **52 female** and **48 male** employees.
- Finance department recorded **0% attrition**, indicating strong employee retention.
- Sales department achieved the highest average performance rating.
- HR department has the highest average salary among all departments.

---

## 💡 Recommendations

- Improve retention in high-attrition departments. 
- Reward top-performing employees.
- Provide regular training and development programs. 
- Maintain gender diversity across teams. 
- Review compensation policies. 
- Focus on employee satisfaction and career growth.
